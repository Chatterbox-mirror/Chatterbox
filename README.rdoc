A prototype for developing mobile apps based on Rails

API: https://gitcafe.com/GitCafe/Chatterbox/wiki

Installation

Prerequisite

Ruby >= 2.0.0
Git

	git clone git://gitcafe.com/ShiningRay/chatterbox-rails.git
	bundle
	bundle exec rake db:setup
	bundle exec rails server


http://localhost:3000/admin to access admin panel
default admin: chatterbox@gitcafe.com password: chatterbox

http://localhost:3000/
